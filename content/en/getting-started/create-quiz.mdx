# Create Your First Quiz

## Steps to create

<Steps>
### Click create

On the dashboard, click the **Create quiz** button to start the creation flow.

### Choose a starting point: blank or template

- **Start from blank**: Create a quiz with no questions, fully customizable.
- **Start from a template**: Pick an existing structure from the template library, clone it, then modify — great for a quick start.

### Choose a quiz type

Pick one of the four types:

- **Quiz**: Fixed order, scoring, and right/wrong judgment.
- **Scorecard**: Multi-dimensional analysis and conditional logic.
- **Outcome Quiz**: Choices vote for outcome types — produces a "which type are you" result.
- **Random Quiz**: Pulls random questions from a bank on each submission.

See [The Four Quiz Types](/en/getting-started/quiz-types) for details.

### Enter the editor

After creation you enter the visual editor and start adding questions. See [Editor & Question Types](/en/editor).
</Steps>

<Callout type="warning">
  Once chosen, the quiz type **cannot be changed**. If you pick the wrong one, you must create a new quiz. Be clear about your use case before creating.
</Callout>

## Templates and question banks

- **Templates**: Reusable blueprints that include a full question structure; clone one and modify it.
- **Question banks**: Reusable collections of questions, commonly used with the "Random Quiz" type.

Managing templates and question banks in depth is an advanced topic covered in later documentation.
