# Dashboard Tour

After signing in you land on the **Dashboard**, which brings together all of your and your team's quizzes. It is the starting point for day-to-day management.

## Main areas

- **Quiz list**: Shows all quizzes in the current team with name, type, status, and more.
- **Create button**: Click it to start [creating a quiz](/en/getting-started/create-quiz).
- **Filter and search**: Quickly locate quizzes by type, status, or keyword.
- **Favorites**: Bookmark frequently used quizzes for quick access.
- **Trash**: Deleted quizzes are kept in the trash and can be restored within the retention period.

## Feature tabs inside a quiz

Once you open a quiz, you typically see these tabs:

| Tab | Purpose |
| --- | --- |
| **Edit** | Build and modify questions, logic, and scoring |
| **Records** | View all submitted responses |
| **Insights** | View charts for completion rate and question-level distribution |
| **Share** | Generate public links, QR codes, social cards, and source tracking |
| **Connect** | Configure webhooks and other external integrations (advanced) |

<Callout type="info">
  The dashboard shows quizzes for the **current team**. If you belong to multiple teams, the list changes when you switch teams.
</Callout>
