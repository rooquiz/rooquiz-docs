# Public Link & QR Code

The public link is the most common way to distribute a quiz. Anyone with the link can open the answer page and fill it in (whether sign-in is required depends on the quiz settings).

## Get and share the link

<Steps>
### Open the share page

In the target quiz, open **Share** → **Link**.

### Copy the public link

The system generates a unique public link for each quiz. Copy it and distribute via chat, email, communities, and other channels.

### Generate a QR code

The same link can produce a **QR code**, handy for posters, offline scenarios, or mobile scan-to-answer.
</Steps>

## Characteristics of the link

- **Unique**: Each quiz has its own public link.
- **No sign-in by default**: Respondents usually fill it in without registering; if the quiz requires identity info or sign-in, it follows those settings.
- **Trackable source**: Combine with [UTM Source Tracking](/en/publish/tracking) to distinguish responses by channel.
