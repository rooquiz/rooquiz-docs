# UTM Source Tracking

When you distribute the same quiz across multiple channels, **UTM source tracking** helps you tell where each response came from, so you can evaluate each channel's effectiveness.

## What is UTM

UTM is a set of parameters appended to a link to label its traffic source. Common ones include:

- **Source**: e.g. `wechat`, `weibo`, `email`.
- **Medium**: e.g. `social`, `banner`, `qrcode`.
- **Campaign**: the name of this campaign, e.g. `spring-launch`.

## How to use it

In the target quiz, open **Share** → **Source tracking**, generate UTM-tagged links for different channels, then distribute each link to its channel.

## Viewing source data

When respondents submit via a UTM-tagged link, the source is recorded with the response. You can distinguish and analyze by source in [Response Records](/en/results/records) and [Insights](/en/results/insights).

<Callout type="info">
  Using **clear, consistent naming** for each channel (e.g. all lowercase, hyphen-separated) keeps your later analysis tidy.
</Callout>
