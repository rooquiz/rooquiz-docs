# Question Types

RooQuiz offers 14+ question types. They are grouped below by purpose to help you pick the right format for each question.

## Choice

| Type | Description | Common uses |
| --- | --- | --- |
| **Single choice** | Pick exactly one option | Single-answer knowledge questions, satisfaction |
| **Multiple choice** | Pick several options | Multi-answer questions, interests/preferences |
| **Dropdown** | Single choice shown as a dropdown | Saves space when there are many options |
| **Cascade** | Multi-level linked selection (e.g. province–city–district) | Region or hierarchical category selection |
| **True/False** | Choose true or false | Yes/no questions |
| **Swiper** | Choose via an image carousel | Image preference, visual options |

## Input

| Type | Description | Common uses |
| --- | --- | --- |
| **Fill-in-the-blank** | Text input | Short answers, open-ended responses |
| **Number** | Numeric input (with validation) | Age, amount, quantity |
| **Date** | Date picker | Birth date, appointment date |
| **Time** | Time picker | Time slot, moment |

## Rating & ordering

| Type | Description | Common uses |
| --- | --- | --- |
| **Rating** | Score on a scale (e.g. 1–5 / 1–10) | Satisfaction, importance |
| **Ordering** | Drag items into order | Priority, preference ranking |

## Content & special

| Type | Description | Common uses |
| --- | --- | --- |
| **Statement** | Displays text only; collects no answer | Section notes, instructions, disclaimers |
| **Random question** | Draws a question from a bank | Used with the "Random Quiz" type |

<Callout type="info">
  Each type has its own settings in the right-hand panel — commonly the title and description, required toggle, option content, and validation rules. Types that support scoring can also set points (see [Scoring & Assessment Rules](/en/editor/scoring)).
</Callout>

<Callout type="warning">
  **Random question** is typically used with the "Random Quiz" type and requires a linked question bank; it may not apply to other types.
</Callout>
