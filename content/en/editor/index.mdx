# Editor Overview

The visual editor is the core workspace for building quizzes. Here you add questions, reorder them, set logic and scoring, and preview exactly what respondents will see in real time.

## Interface layout

The editor is usually divided into three areas:

- **Question type panel**: Choose the type to add; click or drag it into the question list.
- **Question editing area**: The body of the quiz, showing and editing all questions; supports **drag-to-reorder** and **duplicating questions**.
- **Settings / properties panel**: Configure details for the selected question or the whole form (options, required toggle, points, etc.).

## Common actions

- **Add a question**: Pick a type from the panel. See the full list in [Question Types](/en/editor/question-types).
- **Reorder**: Drag questions to change their order.
- **Duplicate**: Quickly copy a similar question and adjust it.
- **Rich text and media**: Question descriptions support rich text and can include images and other media.
- **Live preview**: Preview the respondent's view at any time.

## Going further
