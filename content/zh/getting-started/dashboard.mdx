# 控制台概览

登录后进入**控制台(Dashboard)**,这里集中展示你与团队的全部测验,是日常管理的起点。

## 主要区域

- **测验列表**:展示当前团队下的所有测验,显示名称、类型、状态等信息。
- **新建入口**:点击新建按钮即可开始 [创建测验](/zh/getting-started/create-quiz)。
- **筛选与搜索**:按类型、状态或关键词快速定位目标测验。
- **收藏夹**:将常用测验加入收藏,便于快速访问。
- **回收站**:被删除的测验会暂存于回收站,可在保留期内恢复。

## 单个测验的功能入口

进入某个测验后,通常包含以下标签页:

| 入口 | 作用 |
| --- | --- |
| **编辑** | 搭建与修改题目、逻辑、评分 |
| **答卷记录** | 查看所有提交的作答 |
| **数据洞察** | 查看完成率与题目维度的图表分析 |
| **分享** | 生成公开链接、二维码、社交分享卡片与来源追踪 |
| **集成 / 连接** | 配置 Webhook 等外部集成(进阶) |

<Callout type="info">
  控制台展示的是**当前所在团队**的测验。如果你属于多个团队,切换团队后看到的列表会随之变化。
</Callout>
