# 题型总览

RooQuiz 提供 14 种以上题型。下面按用途分组介绍,帮助你为每道题挑选合适的形式。

## 选择类

| 题型 | 说明 | 常见用途 |
| --- | --- | --- |
| **单选** | 一组选项中只能选一个 | 单选知识题、满意度单选 |
| **多选** | 一组选项中可选多个 | 多选题、兴趣偏好 |
| **下拉** | 以下拉列表呈现的单选 | 选项较多时节省空间 |
| **级联** | 多级联动选择(如省-市-区) | 地区、分类等层级选择 |
| **判断** | 对 / 错 二选一 | 是非题 |
| **轮播** | 以图片轮播方式选择 | 图片偏好、视觉化选项 |

## 输入类

| 题型 | 说明 | 常见用途 |
| --- | --- | --- |
| **填空** | 文本输入 | 简答、开放性回答 |
| **数字** | 数值输入(可校验) | 年龄、金额、数量 |
| **日期** | 日期选择器 | 出生日期、预约日期 |
| **时间** | 时间选择器 | 时间段、时刻 |

## 评价与排序类

| 题型 | 说明 | 常见用途 |
| --- | --- | --- |
| **评分** | 按量表打分(如 1–5 / 1–10) | 满意度、重要性评分 |
| **排序** | 拖拽对若干项排序 | 优先级、偏好排序 |

## 内容与特殊类

| 题型 | 说明 | 常见用途 |
| --- | --- | --- |
| **说明** | 仅展示文字/说明,不收集作答 | 分节说明、指引、声明 |
| **随机题** | 从题库随机抽取题目 | 配合「随机题库」类型 |

<Callout type="info">
  每种题型在右侧设置面板中都有各自的配置项,常见包括:题目标题与描述、是否必答、选项内容、校验规则等。带评分能力的题型还可设置分值(详见 [评分与测评规则](/zh/editor/scoring))。
</Callout>

<Callout type="warning">
  **随机题**通常用于「Random Quiz 随机题库」类型,需要先关联题库;在其他类型中可能不适用。
</Callout>
