# Welcome to RooQuiz Docs

This documentation site is built with [Nextra](https://nextra.site) and the
**docs theme**, running on Next.js App Router.

## Getting started

- Run `pnpm dev` to start the development server.
- Edit files under the `content/` directory to add or change pages.
- Use `_meta.js` files to control the sidebar order and titles.

Check out the [About](/about) page for more.
